# Superforecasting 概率校準框架

## 來源
- **Philip Tetlock**《Superforecasting: The Art and Science of Prediction》
- **Nate Silver**《The Signal and the Noise》

---

## 核心理念

> **超級預測者不是先知，他們只是在校準上比普通人好一點。**

超級預測者的特點：
- 持續更新信念（貝葉斯思維）
- 用概率而非確定性表達
- 區分信號與噪聲
- 承認不確定性

## Outside-In 方法

### 三步框架

```
┌───────────────────┐
│ 1️⃣ Base Rate     │  "歷史上這類事件發生率是多少？"
│    (外部視角)      │
└─────────┬─────────┘
          ↓
┌───────────────────┐
│ 2️⃣ 具體因素調整   │  "這次有什麼特殊？"
│    (內部視角)      │
└─────────┬─────────┘
          ↓
┌───────────────────┐
│ 3️⃣ 貝葉斯更新    │  "新信息如何改變概率？"
│    (持續迭代)      │
└───────────────────┘
```

### Step 1: 找 Base Rate

**永遠從外部視角開始**。

問自己：
- "在所有類似的情況中，這種結果發生的頻率是多少？"
- "參考類別（reference class）是什麼？"

**範例**：
```
問題：某個新創公司會成功 IPO 嗎？

❌ 內部視角："他們的團隊很強、技術很好..."
✅ 外部視角："歷史上，種子輪公司成功 IPO 的比例是 ~1%"

Base Rate = 1%
```

### Step 2: 具體因素調整

在 base rate 基礎上，根據**具體情況**調整。

**調高因素**：
- 這個團隊有成功退出經歷
- 市場正在快速增長
- 有獨特的技術壁壘

**調低因素**：
- 市場競爭激烈
- 資金快燒完了
- 關鍵人員離職

**調整原則**：
- 每個因素的調整要有根據
- 避免過度調整（人類傾向於從 base rate 調整太多）
- 記錄每個調整的理由

### Step 3: 貝葉斯更新

當新信息出現時，更新概率。

**貝葉斯公式**：
```
P(A|B) = P(B|A) × P(A) / P(B)

其中：
- P(A|B) = 新的概率（後驗）
- P(A) = 原來的概率（先驗）
- P(B|A) = 如果 A 為真，看到 B 的概率（似然度）
- P(B) = 看到 B 的概率
```

**簡化版思考**：
```
新概率 = 舊概率 × 更新因子

如果新信息支持假設：更新因子 > 1
如果新信息反對假設：更新因子 < 1
如果新信息無關：更新因子 = 1
```

## Brier Score：校準度量

### 公式
```
Brier Score = (預測概率 - 實際結果)²

實際結果 = 1（發生）或 0（未發生）
```

### 解讀

| Brier Score | 等級 |
|-------------|------|
| 0.00 | 完美預測 |
| < 0.10 | 優秀 |
| < 0.15 | 超級預測者水平 |
| 0.25 | 隨機猜測（50/50）|
| > 0.25 | 比隨機還差 |

### 範例計算
```
預測：某事件有 70% 會發生
結果：事件發生了

Brier = (0.70 - 1)² = 0.09 ✅

預測：某事件有 70% 會發生
結果：事件沒發生

Brier = (0.70 - 0)² = 0.49 ❌
```

### 追蹤表格
```
| 日期 | 市場 | 預測 | 結果 | Brier |
|------|------|------|------|-------|
| 2/20 | 選舉 A | 65% | 1 | 0.12 |
| 2/21 | 法案 B | 30% | 0 | 0.09 |
| 2/22 | 比賽 C | 80% | 0 | 0.64 |
| --- | 平均 | --- | --- | 0.28 |
```

## 信號 vs 噪聲 (Nate Silver)

### 定義
- **信號**：有真實預測價值的信息
- **噪聲**：隨機波動，沒有意義

### 常見噪聲來源
| 來源 | 為什麼是噪聲 |
|------|-------------|
| 單日民調變化 | 抽樣誤差 |
| 短期價格波動 | 隨機交易 |
| 個別專家意見 | 專家也常常錯 |
| 最新消息 | 可能是假消息或過度反應 |

### 區分方法
```
問自己：

1. 這個信息有多少獨立來源確認？
2. 這個變化在歷史模式中是正常的嗎？
3. 這個信息能被其他數據解釋嗎？
4. 等 24 小時後這個信息還重要嗎？
```

## 校準練習

### 個人校準測試

給自己 10 個問題，每個都給出概率，然後檢查結果：

```
如果你說 70% 的 10 件事，應該有 ~7 件發生
如果你說 90% 的 10 件事，應該有 ~9 件發生
```

**校準圖**：
```
理想校準（對角線）：
100%|          ●
 90%|        ●
 80%|      ●
 70%|    ●
 60%|  ●
 50%|●
    +-------------------
     50% 60% 70% 80% 90% 100%
         實際發生率
```

### 常見校準錯誤

**過度自信**：
- 說 90%，但實際只有 70% 發生
- 解決：把你認為的 90% 改成說 75%

**過度保守**：
- 說 60%，但實際有 80% 發生
- 解決：更信任自己的判斷

## 超級預測者的習慣

### 每日習慣
1. 檢查持有預測的新信息
2. 問自己："有什麼能改變我的想法？"
3. 更新概率（哪怕是小幅度）

### 每週習慣
1. 回顧本週預測的表現
2. 計算 Brier Score
3. 尋找系統性偏差

### 每月習慣
1. 分析整體校準
2. 調整個人偏差修正
3. 學習新的 base rate

## 認知偏誤警示

| 偏誤 | 描述 | 預測中的表現 | 對策 |
|------|------|-------------|------|
| 可得性偏誤 | 容易想起的事件被高估 | 高估近期/戲劇性事件概率 | 查 base rate |
| 錨定效應 | 被初始數字影響 | 被市場價格錨定 | 先寫預測再看市場 |
| 確認偏誤 | 只找支持觀點的證據 | 忽略反面信息 | 主動找反證 |
| 後見之明 | "我早就知道" | 無法從錯誤中學習 | 記錄預測時的理由 |
| 代表性偏誤 | 過度依賴典型性 | 忽略 base rate | Outside-In 方法 |
