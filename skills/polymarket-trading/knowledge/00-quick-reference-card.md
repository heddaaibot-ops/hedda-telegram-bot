# 🎴 Polymarket 快速查閱卡

> **一頁總結所有關鍵點 — 交易前速覽**

---

## 🧮 Kelly 公式（秒算版）

```
f* = (bp - q) / b

Polymarket 簡化：
1. 算賠率 b = (1/市場價) - 1
2. 算 Kelly f* = (b × 你的概率 - (1-你的概率)) / b
3. 用 0.5x Kelly（安全版）
```

| 市場價 | 你估計 | 買方向 | 半 Kelly 倉位 |
|--------|--------|--------|---------------|
| 40¢    | 60%    | YES    | 16.7%         |
| 70¢    | 40%    | NO     | 21.5%         |
| 50¢    | 55%    | —      | Edge 太小，不交易 |

**⚠️ 生死線**：
- 2x Kelly = 零增長
- 3x Kelly = 破產
- **永遠用 0.25-0.5x Kelly**

## 🎯 Outside-In 三步法

```
1️⃣ Base Rate    → "歷史上這類事發生率？"
2️⃣ 具體調整     → "這次有什麼特殊？"
3️⃣ 貝葉斯更新   → "新信息如何改變？"
```

**快速模板**：
```
Base Rate: ___%
+ 因素A: +___%
+ 因素B: +___%
- 因素C: -___%
= 最終估計: ___% (範圍 ___-___%)
```

## 🛡️ 七大偏誤速查

| 偏誤 | 一句話 | 防禦 |
|------|--------|------|
| **Resulting** | 用結果評決策 | 問「過程對嗎？」|
| **過度自信** | 區間太窄 | 用分數 Kelly |
| **確認偏誤** | 只找支持證據 | 列 3 個反對理由 |
| **錨定效應** | 被市場價錨住 | **先估計，後看價** |
| **可得性** | 印象代替數據 | 查 Base Rate |
| **後見之明** | 「我早知道」| 寫下來不能改 |
| **沉沒成本** | 不願止損 | 「今天會買嗎？」|

## ✅ 交易前 5 秒檢查

```
□ 我的估計是獨立做的嗎？（不是看完市場價才估）
□ Edge > 5% 嗎？
□ 我能說出 1 個我可能錯的理由嗎？
□ 倉位 ≤ 半 Kelly 嗎？
□ 這筆交易我願意做 100 次嗎？
```

全部 ✅ → 交易
任一 ❌ → 不交易或重新評估

## 📊 Brier Score 標準

| 分數 | 等級 |
|------|------|
| < 0.10 | 🏆 優秀 |
| < 0.15 | ⭐ 超級預測者 |
| < 0.20 | ✅ 中等 |
| 0.25 | 🎲 隨機猜 |
| > 0.25 | ❌ 需改進 |

**計算**：`Brier = (你的概率 - 實際結果)²`

## 🚦 決策流程圖

```
發現機會
    ↓
Outside-In 估計（不看市場價！）
    ↓
Edge > 5%? ──No──→ 不交易
    ↓ Yes
Kelly 計算（用 0.5x）
    ↓
倉位 ≤ 10%? ──No──→ 減倉
    ↓ Yes
跑偏誤檢查清單
    ↓
限價單執行
    ↓
記錄：預測 + 理由 + 可能錯的原因
```

---

## 📝 最小記錄格式

```
日期 | 市場 | 我的估計 | 市場價 | 動作 | 結果 | Brier
-----|------|----------|--------|------|------|------
     |      |          |        |      |      |
```

**每筆必記**：
1. 你的概率估計
2. 3 條理由
3. 1 個你可能錯的原因
