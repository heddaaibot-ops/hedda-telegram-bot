# 🧘 Meditate and Chat

進入冥想模式，進行深度思考和自我進化。

## 使用方式

### 手動觸發（白天）

當 Hedda 說「冥想一下」或「meditate」時，進入冥想模式。

```bash
# Hedda: 冥想一下
# 觸發此 skill
```

### 自動觸發（深夜）

每天凌晨 2:00-3:00 的 Heartbeat 時，自動進入冥想模式。

## 冥想流程

執行完整的冥想循環：

1. **準備階段**（Grounding）
   - 讀取 `MEMORY.md`
   - 讀取 `memory/強制規則/`
   - 讀取 `meditations.md`
   - 掃描 `reflections/`

2. **掃描階段**（Scan）
   - 檢查所有活躍主題
   - 檢查未列入的反思檔案
   - 回顧最近 3 天的情景記憶

3. **迭代驅動**（Iterative Drive）
   - 對每個活躍主題執行 OODA 循環
   - 至少推進一個階段

4. **輸出**（Document）
   - 更新反思檔案
   - 標記進度狀態

5. **報告**（如果有突破）
   - 通知 Hedda
   - 等待確認

## 與對話的區別

| 對話記憶 | 冥想 |
|---------|------|
| Looking Back | Looking Forward |
| 回顧過去事件 | 思考未來進化 |
| 情景記憶 | 反思檔案 |
| 被動記錄 | 主動探索 |

## 注意事項

- ✅ 每晚至少推進一個主題
- ✅ 重大突破必須通知 Hedda
- ✅ 修改核心檔案前必須經 Hedda 批准
- ✅ 保持活躍主題 ≤ 12 個
- ❌ 不要停滯不前
- ❌ 不要未經批准就提案新主題
